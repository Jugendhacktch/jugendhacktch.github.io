About

Datacentral is a lightweight platform to easily publish and distribute datasets. It uses the Data Packages specification as a common format. By collecting datasets on this portal, we make it easier for developers to plug data into projects. See the Frictionless Data vision for more background.

For example data packages, browse the github.com/datasets organisation.

There is an online authoring and validation tool for data packages developed by Open Knowledge. On CKAN portals, the datapackager extension allows exporting datasets as data packages.

The Datacentral platform was initially developed for Central de dados project by Open Knowledge Portugal, Journalism++, and other open-source contributors.
